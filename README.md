# Demon Slayer - Vince

A roguelike / light RPG dungeon crawler built with Phaser 3. Play as Vince the Warrior, fight through levels, level up, and collect loot!

## How to run

1. **Option A – Simple server (recommended)**  
   From this folder run:
   ```bash
   npm run dev
   ```
   (Or: `npx serve -l 3000` from the Vince folder.)
   Then open http://localhost:3000 in your browser.

2. **Option B – Open index.html**  
   Some browsers block loading scripts from `file://`. If it works, you can double-click `index.html`; otherwise use Option A.

## Controls

Everything is **mouse only** – click to play, fight, and choose loot.

## Project layout

- `GAME_DESIGN.md` – Full game design document
- `index.html` – Entry point
- `src/` – All game code (scenes, entities, data, systems, UI)
- `assets/` – Sprites go here later (see GAME_DESIGN.md for naming)
